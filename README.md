# ReactSetup (Now with a webpacked node server for creating an api)

**Use for prototyping**

## Features

- React with Webpack and Babel ready to go
- Express server
- Auto reload server

## Setup

1. `npm i`
2. `npm run dev`
3. Server runs at port 3001
